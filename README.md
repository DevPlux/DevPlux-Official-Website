# DevPlux-Official-Website
The official website repository of DevPlux, showcasing our services, projects, company profile, and latest updates. Built with modern web technologies, focused on performance, scalability, and a clean user experience.

![Cover_Image](cover/img.png)

# Installation

If pnpm is not installed, first install it via npm:

```bash
npm install -g pnpm
```

### Navigate to the project directory

```bash
cd frontend
```

### Install dependencies

```bash
pnpm install
```

### Start the development server

```bash
pnpm dev or pnpm run dev
